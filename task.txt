HW Разработка собственного проекта. Возможные темы:
ToDo List Manager
Personal Book Library
Film Reviews Site
My Budget Manager
Calories Manager
Online shop
Chat Rooms

Требование к проекту:
* Должны быть использованы следующие технологии: Git, Maven, Spring, Tomcat, RDBMS
* Минимум 2 доменных сущности (две таблицы сущностей)
* Минимум одна связь OneToMany
* Полноценный CRUD для одной из сущностей
* JDBC or JPA
* Минимум 3 разных JSP страницы
* Минимум 1 полноценная форма с валидацией входящих параметров
* Должен быть реализован пейджинг, выбор сортировки хотя бы по одному параметру, фильтр хотя бы по одному параметру
* Авторизация, если логика сайта требует
* Проект должнен быть законченным продуктом, которым можно пользоваться клиентам. Запускаться и работать без ошибок

(Ачивки) Дополнительные баллы можно заработать, если будет присутствовать:
* (Domain master) 4+ доменных сущностей. Минимум 3 связи OneToMany.
* (JSP master) 5 уникальных JSP страниц
* (Data manipulation master) Страница со сложным фильтром, пейджингом, сортировками. Суммарно - 6 параметров
* (Spring Security master) Полноценное использование 3+ разных ролей на сайте
* (View Design master) Сложные html страницы, несколько блоков на одной странице. Много стилей.
* (Testing DAO master) Все методы DAO покрыты полноценными тестами
* (Testing Controller master) Все Spring контроллеры покрыты полноценными тестами
